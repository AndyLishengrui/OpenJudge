# Print Graphics Problerm (II)
Time Limit: 1 Sec  Memory Limit: 2 MB


## Description
向标准输出上打印一些用ASCII字符组成的图形。

## Input
输入为多个整数n，0<n<100。当n为0时结束输入。

## Output
若n为偶数，则输出一个正向的n层等腰三角形；n为奇数，则输出一个倒向的n层等腰三角形。三角形由“+”组成。任意两个图形之间有一个空行分隔，格式见sample。

## Sample Input
```
5
4
0
```
## Sample Output
```
+++++++++
 +++++++
  +++++
   +++
    +

   +
  +++
 +++++
+++++++

```
