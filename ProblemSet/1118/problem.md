# 计算a-b之间特定数值之和
Time Limit: 1 Sec  Memory Limit: 16 MB


## Description
求整数a~b之间，含有某个特定数字k的数值之和sum。假定a、b、k都是整数，且a<=b，0<=k<10。b-a<100000，sum在int类型的表示范围之内。


## Input
输入为多行。第一行M表示有M组输入。之后有M行，每一行含有3个值：a,b,k，分别表示数据的范围和特定的数字。


## Output
输出为多行，每一行对应于每一组输入的结果sum。


## Sample Input
```
2
1 100 3
-10 10 2
```
## Sample Output
```
792
0
```

## HINT
注意对负数的处理。
