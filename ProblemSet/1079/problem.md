# 是否构成三角形？
Time Limit: 1 Sec  Memory Limit: 32 MB


## Description
﻿给出三个整数，代表三条边的长度，判断这三条边的长度是否能构成一个三角形？


## Input
第一行是n(1<=n<=100)，表示有n组测试数据
接下来有n行，每一行包含三个整数，表示三个边长（1<=边长<=100）


## Output
如果三条边能构成三角形，输出YES，否则输出NO


## Sample Input
```
3
1 2 3
2 8 7
20 20 1
```
## Sample Output
```
NO
YES
YES
```
