# 打印金字塔 之一
Time Limit: 1 Sec  Memory Limit: 16 MB


## Description
输出由数字字符组成的图形。


## Input
输入为一个正整数N，不超过int类型的表示范围。


## Output
输出由N行数字字符组成的等腰三角形，其中第i（0<i<=N）行的数字是：
（1）如果0<i<10，即为i。
（2）如果i>9，则输出i的个位数。


## Sample Input
```
5
```
## Sample Output
```
    1
   222
  33333
 4444444
555555555
```
