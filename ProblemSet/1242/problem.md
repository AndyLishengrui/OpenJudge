# 二进制整数转十进制
Time Limit: 1 Sec  Memory Limit: 2 MB


## Description
给出一个二进制的非负整数x，x<232，把它转换成十进制数输出。



## Input
输入为多行，每行一个二进制非负整数x。


## Output
每行输出x对应的十进制数值。


## Sample Input
```
0
1
01
10
11
100001
1111111111111111
```
## Sample Output
```
0
1
1
2
3
33
65535

```

## HINT
 注意数据范围！！！
