# 求指定数字的和
Time Limit: 1 Sec  Memory Limit: 128 MB


## Description
求所有能被M整除且不含有重复数字的三位数的和。（三位数即区间[100,999]内的数。）


## Input
输入为若干行，每行包含一个正整数M。M在int类型的范围内。


## Output
输出为多行，每行输出与上述每个M一一对应，是所有能被M整除的、不含重复数字的三位数之和。输出格式见样例。


## Sample Input
```
3
5
7
```
## Sample Output
```
case 1:124587
case 2:74720
case 3:51163

```
