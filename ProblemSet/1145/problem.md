# 求累加和
Time Limit: 1 Sec  Memory Limit: 2 MB


## Description
编程求min~max的累加和（含min和max），其中max>=min>0。




## Input
输入为多行。第一行是一个整数N>0，表示后面有N个测试用例。后面有N行，每行包含2个整数，分别是min和max。


## Output
输出为N行，每个测试用例的计算结果占据一行。每行的格式为：
case i:sum=s.
其中i表示测试用例的编号（从1开始），s是该测试用例对应的累加和（设不超过int的表示范围）。


## Sample Input
```
2
1 10
1 100
```
## Sample Output
```
case 1:sum=55.
case 2:sum=5050.

```
