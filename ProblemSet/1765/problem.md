# 数字的千位分隔
Time Limit: 1 Sec  Memory Limit: 128 MB


## Description
把整数利用逗号进行千位分隔并输出。



## Input
输入第一行是整数T(0<T<=50)，表示接下来有T行，每行有一个整数，即为要利用逗号千位分隔的整数k，其中0<=k<=2^64-1。


## Output
对应每一行数据输入，输出一行用逗号进行千位分隔的数字串。



## Sample Input
```
3
12
123
12345
```
## Sample Output
```
12
123
12,345

```
