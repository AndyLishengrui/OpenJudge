# Print Graphics Problerm (V)
Time Limit: 1 Sec  Memory Limit: 2 MB


## Description
向标准输出上打印一些用ASCII字符组成的图形。


## Input
输入为多个奇数n，1<n<100，至文件尾结束。


## Output
输出为多个n层的空心菱形，任意两个图形之间有一个空行分隔，格式见sample。


## Sample Input
```
3
7
```
## Sample Output
```
 *
* *
 *

   *
  * *
 *   *
*     *
 *   *
  * *
   *
```
