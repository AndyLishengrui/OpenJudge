# 夏季促销
Time Limit: 1 Sec  Memory Limit: 128 MB


## Description

商场夏季促销，购物399元（含）以上，95折；购物899元（含）以上，9折；购物2499元（含）以上，85折；购物4899元（含）以上，8折。并且不收取角分部分。

根据购物金额，确定用户实际需要支付的数目。


## Input
输入客户购买若干物品的单价金额，均为正整数，至EOF结束。



## Output
输出用户实际需要支出的数目，以元为单位，保留两位小数。



## Sample Input
```
105
398
742
82
888
276
```
## Sample Output
```
2241.00

```
