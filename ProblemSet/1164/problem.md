# Print Graphics Problerm (I)
Time Limit: 1 Sec  Memory Limit: 2 MB


## Description
向标准输出上打印一些用ASCII字符组成的图形。


## Input
输入为一个整数n，0<n<100。


## Output
输出一个n层等腰三角形，格式见sample。


## Sample Input
```
5
```
## Sample Output
```
    +
   +++
  +++++
 +++++++
+++++++++

```

## HINT
 推导出第i层要输出的空格和“+”与i的关系，用循环很容易完成。
