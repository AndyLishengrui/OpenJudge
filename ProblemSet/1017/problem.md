# 哪一行比较长
Time Limit: 1 Sec  Memory Limit: 2 MB


## Description
读取两行字符串，按每行的长度从长到短输出。


## Input
输入为两行，每行不会超过26个字符。


## Output
输出为两行，按每行的长度从长到短输出。


## Sample Input
```
abcdefghijk
abcdefghijklmnopqrstuvwxyz
```
## Sample Output
```
abcdefghijklmnopqrstuvwxyz
abcdefghijk
```

## HINT
了解字符串的存储和操作，了解gets()和scanf("%s")读入字符串的不同之处。
