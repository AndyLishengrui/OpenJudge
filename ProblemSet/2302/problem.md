# 输出回字形
Time Limit: 1 Sec  Memory Limit: 16 MB


## Description
输出n阶的回字形。定义1、2、3阶的回字形如下。
1阶回字形：
      @@@
      @ @
      @@@
2阶回字形
      @@@@@@@
      @     @
      @ @@@ @
      @ @ @ @
      @ @@@ @
      @     @
      @@@@@@@
3阶回字形
      @@@@@@@@@@@
      @         @
      @ @@@@@@@ @
      @ @     @ @
      @ @ @@@ @ @
      @ @ @ @ @ @
      @ @ @@@ @ @
      @ @     @ @
      @ @@@@@@@ @
      @         @
      @@@@@@@@@@@


## Input
输入一个整数n，n取1～9。


## Output
输出对应的图形。


## Sample Input
```
2
```
## Sample Output
```
@@@@@@@
@     @
@ @@@ @
@ @ @ @
@ @@@ @
@     @
@@@@@@@

```
