# 求字符的值
Time Limit: 1 Sec  Memory Limit: 2 MB


## Description
从键盘输入3个字符（不含双字节字符），分别输出每个字符的十进制值（ASCII码）、八进制值和十六进制值。

## Input
输入为3个字符。

## Output
输出为3行。
每一行为每个字符（对应输入顺序）的十进制、八进制和十六进制值，用空格分隔开。每个输出的值占3个字符，不足3个字符前面补0。


## Sample Input
```
0 A
```
## Sample Output
```
048 060 030
032 040 020
065 101 041

```

## HINT
了解字符值的存储和整型的关系。
